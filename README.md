# Bhupinder Singh Liddar – Portfolio Website

A premium, elegant personal portfolio website for **Bhupinder Singh Liddar**, a retired Canadian diplomat with over 40 years of experience in international relations, media, and public service.

---

## 🌐 Overview

This website showcases:
- Professional biography
- Career milestones
- Media and editorial contributions
- Global diplomatic presence
- Contact and inquiry form

Designed with a **luxury editorial style**, inspired by global institutions like the United Nations and executive personal brands.

---

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- PHP (Contact Form Backend)

---

## 🎨 Features

- ✨ Premium light-themed UI (white + navy + gold)
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🧭 Smooth navigation and scroll animations
- 🕰️ Timeline-based experience section
- 📰 Media and editorial showcase
- 🌍 Global presence visualization
- 📩 Functional contact form using PHP
- ⚡ Fast and lightweight (optimized for shared hosting)
